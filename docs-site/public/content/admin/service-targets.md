# 서비스 목표 관리 (M/M)

## 개요

**서비스 목표 관리** 페이지에서는 각 서비스의 **목표 M/M**(Men/Month)과 **Saved M/M**(절감 실적)을 설정하고, 달성률을 추적할 수 있습니다.

- **목표 M/M**: 이 서비스가 절감할 것으로 기대하는 인력 공수
- **Saved M/M**: 실제로 절감된 인력 공수 실적
- **달성률**: Saved M/M ÷ 목표 M/M × 100%

> 💡 목표 M/M과 Saved M/M의 **모든 변경은 감사 로그에 기록**됩니다.

## 접근 권한

| 역할 | 조회 | 수정 |
|------|------|------|
| SUPER_ADMIN | 전체 서비스 | 전체 서비스 |
| ADMIN | 같은 부서 서비스만 | 같은 부서 서비스만 |
| 일반 사용자 | ✕ | ✕ |

**사이드바**: 시스템 관리 > **서비스 목표 관리**

## 사용 방법

### 1. 목표/실적 설정

1. 서비스 목표 관리 페이지에서 원하는 서비스의 **수정** 버튼 클릭
2. **목표 M/M**과 **Saved M/M** 값을 입력 (소수점 1자리까지)
3. **저장** 버튼 클릭

### 2. 달성률 확인

- 달성률은 **바 차트**로 시각화됩니다
- 색상 기준:
  - 🟢 100% 이상: 초록
  - 🔵 70~99%: 파랑
  - 🟡 40~69%: 주황
  - 🔴 40% 미만: 빨강

### 3. AI 추정 M/M

시스템 LLM이 설정된 경우, 매일 자정(KST)에 **최근 5영업일 평균 DAU**를 기반으로 각 서비스의 M/M 절감 효과를 AI가 자동 추정합니다.

- 클릭하면 AI의 **추정 근거**(reasoning)를 볼 수 있습니다
- 신뢰도: 🟢 HIGH / 🔵 MEDIUM / 🟡 LOW

## 감사 로그

모든 목표/실적 변경은 감사 로그에 `UPDATE_SERVICE_TARGET` 액션으로 기록됩니다.

감사 로그 페이지에서 **"서비스 목표 변경"** 탭을 클릭하면 관련 변경 내역만 필터링하여 볼 수 있습니다.

## 서비스 생성 시 참고

서비스 생성 위자드에서는 목표 M/M을 입력하지 않습니다. 서비스를 먼저 생성한 뒤, 이 페이지에서 목표를 설정하세요.

> ⚠️ 서비스 수정 모달에서도 목표 M/M은 수정할 수 없습니다. 반드시 **서비스 목표 관리** 페이지를 이용하세요.
