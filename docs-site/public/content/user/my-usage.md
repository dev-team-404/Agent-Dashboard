# 사용량 확인

Dashboard에서 본인의 API 사용량을 확인하는 방법을 안내합니다.

## 사용량 페이지 접근

1. Dashboard에 로그인
2. 좌측 메뉴 또는 메인 화면에서 **내 사용량** 클릭

## 기간별 사용량

### 오늘 사용량

오늘 하루 동안의 API 사용 현황을 확인합니다.

| 항목 | 설명 |
|------|------|
| 총 요청 수 | 오늘 호출한 API 횟수 |
| 입력 토큰 | 오늘 사용한 입력(prompt) 토큰 수 |
| 출력 토큰 | 오늘 사용한 출력(completion) 토큰 수 |
| 총 토큰 | 입력 + 출력 총 토큰 수 |

### 이번 주 사용량

이번 주(월~일) 동안의 누적 사용량을 확인합니다. 일별 추이 차트도 함께 표시됩니다.

### 이번 달 사용량

이번 달 1일부터 현재까지의 누적 사용량을 확인합니다. 주별 추이와 전월 대비 증감을 확인할 수 있습니다.

## 모델별 사용량

어떤 LLM 모델을 얼마나 사용했는지 확인할 수 있습니다.

### 확인 항목

| 항목 | 설명 |
|------|------|
| 모델명 | 사용한 LLM 모델 이름 |
| 요청 수 | 해당 모델로 호출한 횟수 |
| 입력 토큰 | 해당 모델에 보낸 입력 토큰 |
| 출력 토큰 | 해당 모델이 생성한 출력 토큰 |
| 비율 | 전체 사용량 대비 비율 |

### 활용 예시

```
모델별 사용량 (이번 달)
━━━━━━━━━━━━━━━━━━━━━━━━━
gpt-4o          ████████████████  62%  (1,234건)
gpt-4o-mini     ████████          31%  (620건)
claude-3.5      ████              7%   (140건)
```

## 서비스별 사용량

어떤 서비스를 통해 API를 호출했는지 확인할 수 있습니다.

### 확인 항목

| 항목 | 설명 |
|------|------|
| 서비스 ID | API를 호출한 서비스 |
| 요청 수 | 해당 서비스를 통한 호출 횟수 |
| 토큰 사용량 | 해당 서비스를 통한 총 토큰 사용량 |

> 여러 서비스를 사용하는 경우 서비스별로 분리된 사용량을 확인하여 어디서 가장 많이 호출하는지 파악할 수 있습니다.

## 최근 사용 이력

최근 API 호출 이력을 시간순으로 확인할 수 있습니다.

### 이력 항목

| 항목 | 설명 |
|------|------|
| 시간 | API 호출 시간 |
| 서비스 | 호출에 사용된 서비스 ID |
| 모델 | 사용된 LLM 모델 |
| 입력 토큰 | 해당 호출의 입력 토큰 수 |
| 출력 토큰 | 해당 호출의 출력 토큰 수 |
| 상태 | 성공/실패 |

### 이력 필터

- **기간 필터**: 특정 기간의 이력만 조회
- **모델 필터**: 특정 모델에 대한 이력만 조회
- **서비스 필터**: 특정 서비스에 대한 이력만 조회
- **상태 필터**: 성공/실패 이력만 조회
