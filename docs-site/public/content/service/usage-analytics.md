# 사용량 분석

서비스의 API 사용 현황을 다양한 관점에서 분석하는 Usage Analytics 기능을 안내합니다.

## Usage Analytics 개요

Usage Analytics는 서비스의 API 호출 데이터를 **개요, 사용자별, 모델별, 부서별** 4가지 탭으로 나누어 분석하는 기능입니다. 메인 대시보드 또는 서비스 상세 화면에서 접근할 수 있습니다.

### 접근 방법

**메인 대시보드에서:**
1. Dashboard 좌측 메뉴에서 **대시보드** 클릭
2. 하단 **Usage Analytics** 영역 확인

**서비스별 분석:**
1. **내 서비스** 페이지에서 서비스 선택
2. 서비스 상세 화면에서 Usage Analytics 확인

> **참고**: 서비스별 분석은 해당 서비스의 데이터만 표시합니다. 메인 대시보드에서는 접근 가능한 전체 데이터를 표시합니다.

## 기간 설정

모든 탭에서 분석 기간을 설정할 수 있습니다.

### 프리셋

| 프리셋 | 기간 |
|--------|------|
| **7일** | 최근 7일 |
| **30일** | 최근 30일 |
| **90일** | 최근 90일 |

### 기간 선택

프리셋 버튼을 클릭하면 해당 기간의 데이터가 자동으로 로드됩니다.

## 개요 탭

전체 사용량을 한눈에 파악할 수 있는 요약 화면입니다.

### 주요 지표 카드

| 지표 | 설명 |
|------|------|
| 총 요청 수 | 선택 기간 내 총 API 호출 횟수 |
| 총 토큰 수 | 선택 기간 내 입력 + 출력 토큰 합계 |
| 입력 토큰 | 선택 기간 내 입력 토큰 합계 |
| 출력 토큰 | 선택 기간 내 출력 토큰 합계 |

### 일별 추이 차트

선택 기간의 일별 사용량 추이를 차트로 표시합니다.

- **X축**: 날짜 (MM-DD)
- **Y축**: 요청 수 / 토큰 수
- 입력 토큰과 출력 토큰을 색상으로 구분

```
일별 사용량 추이 (최근 7일)
━━━━━━━━━━━━━━━━━━━━━━━━━
03-04  ████████████████  12,345 req
03-05  ██████████████████  14,567 req
03-06  ████████████████████  16,789 req
03-07  ██████████████████  15,234 req
03-08  ████████████████  12,890 req
03-09  ████  3,456 req
03-10  ██  1,234 req
```

## 사용자별 탭

사용자별 API 사용량을 분석합니다.

### 표시 항목

| 항목 | 설명 |
|------|------|
| 사용자 ID | API를 호출한 사용자 loginid |
| 요청 수 | 해당 사용자의 API 호출 횟수 |
| 입력 토큰 | 해당 사용자의 입력 토큰 합계 |
| 출력 토큰 | 해당 사용자의 출력 토큰 합계 |
| 총 토큰 | 입력 + 출력 토큰 합계 |

### 사용자별 차트

- 상위 사용자를 막대 그래프로 표시
- 사용자별 토큰 사용량 비율을 시각적으로 확인
- 특정 사용자의 과도한 사용을 빠르게 파악 가능

> **활용**: 사용자별 탭의 데이터를 기반으로 [Rate Limit](/docs/service/rate-limits)을 개별 설정하면, 특정 사용자의 과도한 사용을 효과적으로 통제할 수 있습니다.

## 모델별 탭

LLM 모델별 사용량을 분석합니다.

### 표시 항목

| 항목 | 설명 |
|------|------|
| 모델명 | LLM 모델 이름 |
| 요청 수 | 해당 모델의 API 호출 횟수 |
| 입력 토큰 | 해당 모델의 입력 토큰 합계 |
| 출력 토큰 | 해당 모델의 출력 토큰 합계 |
| 총 토큰 | 입력 + 출력 토큰 합계 |

### 모델별 차트

- 모델별 사용 비율을 차트로 표시
- 어떤 모델이 가장 많이 사용되는지 파악 가능
- 모델 리소스 분배 최적화에 활용

```
모델별 사용 비율
━━━━━━━━━━━━━━━━━━━━━━━━━
gpt-4o          ████████████████████  45%
claude-sonnet   ████████████          28%
gemini-pro      ████████              18%
gpt-4o-mini     ████                   9%
```

## 부서별 탭

부서(dept)별 사용량을 분석합니다.

### 표시 항목

| 항목 | 설명 |
|------|------|
| 부서명 | dept 이름 |
| 요청 수 | 해당 부서의 API 호출 횟수 |
| 입력 토큰 | 해당 부서의 입력 토큰 합계 |
| 출력 토큰 | 해당 부서의 출력 토큰 합계 |
| 총 토큰 | 입력 + 출력 토큰 합계 |

### 부서별 차트

- 부서별 사용 비율을 차트로 표시
- 조직 내 LLM 사용 분포를 한눈에 파악
- 부서별 비용 분배 기초 자료로 활용

## CSV 내보내기 (Export)

모든 탭의 데이터를 CSV 파일로 내보낼 수 있습니다.

### 내보내기 방법

1. 원하는 탭(개요/사용자별/모델별/부서별) 선택
2. 기간 설정
3. 우측 상단 **다운로드** 아이콘 클릭
4. CSV 파일이 자동 다운로드됨

### CSV 파일 형식

- **인코딩**: UTF-8 (BOM 포함, Excel에서 한글 정상 표시)
- **구분자**: 콤마(,)
- **파일명**: `{탭명}.csv` (예: `overview.csv`, `by-user.csv`)

### 내보내기 데이터 예시 (사용자별)

```csv
사용자ID,요청수,입력토큰,출력토큰,총토큰
hong.gildong,1234,456789,123456,580245
kim.chulsoo,567,234567,98765,333332
lee.younghee,890,345678,134567,480245
```

### 활용 방법

- Excel/Google Sheets에서 추가 분석
- 부서별 비용 보고서 작성
- 사용량 추이 장기 보관
- 경영진 보고 자료 작성
