# 사용자/권한 관리

Dashboard에서 사용자 목록을 확인하고 권한을 관리하는 방법을 안내합니다.

## 사용자 목록 확인

### 목록 조회

1. Dashboard 좌측 메뉴에서 **사용자 관리** 클릭
2. dept 내 사용자 목록이 표시됩니다.

### 사용자 정보

각 사용자에 대해 아래 정보를 확인할 수 있습니다.

| 항목 | 설명 |
|------|------|
| 사용자 ID | 사용자 고유 식별자 |
| 이름 | 사용자 이름 |
| 부서(dept) | 소속 부서 |
| 역할 | USER 또는 ADMIN |
| 최근 활동 | 마지막 API 호출 시간 |

### 검색 및 필터

- **검색**: 사용자 ID 또는 이름으로 검색
- **필터**: 역할(Admin/User)별 필터링 가능

## Admin 지정

일반 사용자에게 Admin 권한을 부여할 수 있습니다.

### 지정 절차

1. **사용자 관리** 페이지에서 대상 사용자 찾기
2. 해당 사용자의 **역할** 항목 클릭
3. **ADMIN**으로 변경
4. **저장** 클릭

### Admin이 할 수 있는 것

Admin으로 지정된 사용자는 아래 기능을 사용할 수 있습니다.

- 서비스 등록, 수정, 삭제
- LLM 모델 등록, 수정, 삭제
- 사용자 권한 관리
- 통계 조회

## Admin 해제

Admin 권한을 해제하여 일반 사용자로 변경할 수 있습니다.

### 해제 절차

1. **사용자 관리** 페이지에서 대상 Admin 찾기
2. 해당 사용자의 **역할** 항목 클릭
3. **USER**로 변경
4. **저장** 클릭

> **주의**: Admin 권한을 해제하면 해당 사용자가 등록한 서비스와 LLM은 유지되지만, 더 이상 수정/삭제할 수 없게 됩니다.

## 권한 체계

Dashboard의 권한 체계는 아래와 같습니다.

### 역할별 권한 비교

| 기능 | USER | ADMIN |
|------|------|-------|
| 대시보드 접속 | O | O |
| 본인 사용량 확인 | O | O |
| 서비스 등록/관리 | X | O |
| LLM 등록/관리 | X | O |
| 사용자 권한 관리 | X | O |
| dept 내 통계 조회 | X | O |

### 권한 범위

- Admin의 관리 권한은 **자신이 속한 dept 내**로 제한됩니다.
- 다른 dept의 서비스, LLM, 사용자는 관리할 수 없습니다.
- Admin이 등록한 서비스와 LLM은 해당 Admin만 수정/삭제할 수 있습니다.

```
dept: SW혁신팀(S.LSI)
  ├── Admin A
  │     ├── 서비스 1 (Admin A만 수정 가능)
  │     └── LLM 모델 X (Admin A만 수정 가능)
  ├── Admin B
  │     ├── 서비스 2 (Admin B만 수정 가능)
  │     └── LLM 모델 Y (Admin B만 수정 가능)
  └── User C
        └── 본인 사용량 확인만 가능
```
